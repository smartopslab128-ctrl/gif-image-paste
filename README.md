# GIF & Image Paste

どこで見つけた GIF/画像を集めて、いつでもどこでも貼れるデスクトップツールです。

- **常時前面**の小さなウィンドウで、コメント欄や投稿画面の横に置いて使えます（トグルでオフ可）
- **システムトレイ**: 閉じるとトレイに最小化。ダブルクリックで再表示。トレイメニューから表示/終了
- **収集**: 「＋ 貼り付け」でクリップボードから、「ファイルから」でダイアログから、または画像・GIF をドラッグ＆ドロップで追加
- **貼り付け**: カードを**ドラッグ**して他アプリにドロップ。または**クリック**でコピー → 貼り付け先で Ctrl+V

## 必要環境

- Node.js 18+
- Windows / macOS / Linux（Electron 対応）

## セットアップ

```bash
npm install
npm start
```

## 使い方

1. **起動**  
   `npm start` でウィンドウが前面に表示されます。

2. **追加**  
   - **「＋ 貼り付け」**: 画像をコピー（Ctrl+C）した状態でクリック、またはウィンドウにフォーカスして Ctrl+V  
   - **「ファイルから」**: ダイアログで画像・GIF を複数選択して追加  
   - **ドラッグ＆ドロップ**: ブラウザやエクスプローラーから画像・GIF をウィンドウ内のドロップゾーンにドロップ  

3. **貼り付け**  
   - **ドラッグ**: 一覧の画像をドラッグして、ブラウザのコメント欄などにドロップ  
   - **クリック＋Ctrl+V**: 一覧の画像をクリックでコピーし、貼り付け先で Ctrl+V  

4. **削除**  
   カードにマウスを乗せて表示される「×」で削除できます。

## 制限（無料）

- 保存数: 最大 50 枚（今後、課金で増量やクラウド同期を検討可能）

## ビルド（配布用 exe）

```bash
npm run build
# または Windows のみ
npm run build:win
```

`dist/` にインストーラーが出力されます。

## 技術

- Electron
- 保存先: ローカル（`userData/library`）、Index は JSON、画像はファイルで保存
- サーバー不要（最小構成）
